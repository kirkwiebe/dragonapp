Welcome to DragonApp!

DragonApp utilizes the API from www.dnd5eapi.co/ and presents the information in an easy-to-consume format. This is also a project for an engineering apprenticeship. The project requires a single-page webapp that pulls data from a web service (or services), renders some kind of visualization of the data, and allows for at least one point of interaction with the data.
